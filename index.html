Projects

Fridge

What's in your fridge? This app was made to answer the question, what can you make with the food in your fridge. The user enters a couple ingredients and the app returns a list of recipes.
This app was made using Node.js, ExpressJS, jQuery, and a Sequelize database for the back end. Materialized was used to style the front end.

ComingUp

ComingUp is the answer that every Outreach coach a General Assembly needs. This is a one stop shop for administrators and students to complete their outreach goals.
ComingUp is a Ruby on Rails app with a strong database and styled with bootstrap. There is also an up-to-date api that calls current events from Meetup.com

Theremin

This is exactly how it sounds. A small app made with Node.js and ExpressJS. It is played like a Theremin.

Ensemble News

Have you ever wanted the full story on current events? Ensemble news searches mulitple apis for current events and gives you every journalists take on the story. This is a MEAN stack app that uses the New York Times Api, The Guardian Api, and more. No need to press search, the app is watching.

Chutes and Ladders

Chutes and Ladders, the website board game. This is a front end app made with HTML, CSS, and jQuery. It was the first project for General Assembly WDI08.

DedElectric

DedElectric is Seattles new favorite rock and roll band you may never have heard of. This website is built before I had any knowledge of coding and uses the godaddy.com website builder.
